## 项目概述

### 项目背景

OpenClaw 是一个基于 AI 的智能助手系统，支持多用户隔离、隐私保护、自动化任务执行等功能。本项目旨在构建一个安全、可靠、可扩展的多用户 AI 助手平台。

### 核心目标

- ✅ 实现多用户隔离和隐私保护
- ✅ 提供自动化新闻采集和发布能力
- ✅ 建立完善的监控告警系统
- ✅ 开发可视化管理界面
- ✅ 实现 Gateway 自动恢复机制

#### 技术突破

**Asset ID 计算算法**:
```javascript
function canonicalStringify(obj) {
  if (obj === null || obj === undefined) return 'null';
  if (typeof obj !== 'object') return JSON.stringify(obj);
  if (Array.isArray(obj)) {
    return '[' + obj.map(item => canonicalStringify(item)).join(',') + ']';
  }
  const sortedKeys = Object.keys(obj).sort();
  const pairs = sortedKeys.map(key => `"${key}":${canonicalStringify(obj[key])}`);
  return '{' + pairs.join(',') + '}';
}

function computeAssetId(obj) {
  const { asset_id, ...rest } = obj;
  const canonical = canonicalStringify(rest);
  return `sha256:${crypto.createHash('sha256').update(canonical).digest('hex')}`;
}
```

症状：用户消息未正确加载记忆

解决：
```bash
# 检查用户目录
ls -la ~/.openclaw/workspace/memory/users/

ls -la ~/.openclaw/workspace/memory/sessions/

# 重启 gateway
openclaw gateway restart
```

**问题 2: 新闻采集数量为 0**

症状：采集完成但无新闻

解决：
```bash
# 检查网站可访问性
curl -I http://www.chinawuliu.com.cn/

# 测试采集器
python3 test_collect_real.py

# 查看日志
tail -f logs/cron.log
```

**问题 3: 监控告警未发送**

症状：触发告警但未收到飞书消息

解决：
```bash
# 检查飞书配置
cat ~/.openclaw/config.json | grep -A 5 "feishu"

# 查看告警日志
tail -f ~/.openclaw/workspace/logs/alert.log

# 测试飞书 API
node monitor.js check
```

**问题 4: Web 界面无法访问**

症状：浏览器无法连接

解决：
```bash
# 检查服务是否运行
ps aux | grep server.js

# 检查端口占用
lsof -i :3456

# 使用其他端口
WEB_ADMIN_PORT=8080 node server.js
```
