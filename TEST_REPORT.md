## 📝 已知问题

### 问题 1: Session 实际创建延迟

**现象**: 子代理信息记录到 pending 队列，但实际 session 未立即创建

**原因**: `sessions_spawn` 需要在 Gateway 主流程中调用

**解决方案**: 
- 修改 Gateway 消息处理流程
- 在收到消息时检查 pending 队列
- 自动创建 session

**优先级**: P0

---

### 问题 2: 配额统计不完整

**现象**: token 使用量未实时统计

**原因**: 需要集成到 token 计数器

**解决方案**:
- 在 token 使用时调用 `quotaManager.useQuota()`
- 定期同步使用量

**优先级**: P1
