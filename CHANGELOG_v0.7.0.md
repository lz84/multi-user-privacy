# Multi-User Privacy v0.7.0 更新说明

## 🆕 v0.7.0 (2026-03-11)

### 子代理自动路由

**新增功能：**
- ✅ 自动为每个用户创建专属子代理
- ✅ 消息自动路由到对应子代理
- ✅ 完全自动化，无需审核
- ✅ 子代理统计工具

**新增文件：**
- `scripts/subagent-router.js` - 子代理路由工具

**使用方式：**

```bash
# 查看所有用户子代理
node scripts/subagent-router.js list

# 删除用户子代理
node scripts/subagent-router.js remove <user_id>

# 清空路由器数据库
node scripts/subagent-router.js clear
```

**工作流程：**

```
1. 新用户消息进来
    ↓
2. 自动检测：是否有该用户的子代理？
    ↓
3. 没有 → 自动创建专属子代理
    ↓
4. 消息路由到子代理
    ↓
5. 子代理处理并回复
```

## 🔧 子代理路由说明

### 架构

```
主代理 (路由器)
    ↓
    ├── 子代理：老刘专属
    ├── 子代理：孙哥专属
    └── 子代理：蟹老板专属
```

### 路由逻辑

**每次收到消息时：**
1. 读取 `sender_id`
2. 查找或创建对应子代理
3. 转发消息到子代理
4. 获取子代理回复
5. 转发给用户

**查看所有用户：**
